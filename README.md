# Local Community Portal

## Project Overview
A full stack web application developed as part of the Web Technologies module.
The platform acts as a local community hub where users can explore events,
activities and services across different community areas.

## Technologies Used
- JavaScript
- Node.js
- Express
- EJS
- SQLite (better-sqlite3)
- HTML & CSS
- AJAX
- Git & GitHub

## Key Features
- Server-side routing using Express
- Dynamic page rendering with EJS templates
- Database-driven content using SQLite
- Client-side and server-side JavaScript logic
- AJAX-based search and data retrieval
- Contact form with data stored in the database

## What I Learned
- Building a full stack web application using JavaScript
- Separating client-side and server-side responsibilities
- Working with relational databases in a Node environment
- Applying good web development and accessibility practices

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `node index.mjs`
4. Open `http://localhost:5000` in your browser
