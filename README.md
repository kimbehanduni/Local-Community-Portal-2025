# Local Community Portal

Community portal featuring four areas (Sports, Health, Education, Arts & Culture), a Featured Event, live search (AJAX), Area pages with type filter (AJAX), FAQ and Contact (stored in SQLite). Basic accessibility (skip link, visible focus) and client/server validation.

## Requirements
- Node.js 20 LTS + npm
- SQLite (the project already includes `db/database.sqlite`)

## Install

npm install
